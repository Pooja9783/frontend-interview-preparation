Sticky Position

an element with positioned : sticky, is based on the user's scroll position.

Static Positions

Static Position is positioned default. it is not affected by the top, bottom, left, right.

Relative Position

An element with Position: relative is positioned relative to normal position.

Absolute Position

An element with Position: absolute is positioned relative to the nearest ancestor .
-If an absolute positioned element has no positoned ancestor, it uses the document body and move along with page scrolling.
Note: absolute positioned elements are removed from the normal flow and can overlap elements.

Fixed Position

an element with positioned : fixed, is relative to the viewport, which means it always stays in the same place even if the page is scrolled.
